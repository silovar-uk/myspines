# myspines

> 段落を動かしながら、長い文章を書くためのローカルファーストエディタ。

myspinesは、本文・番号付きアウトライン・メモを一つの画面で扱う長文執筆ツールです。
UI v4.4では、タイトル・各見出し・本文の文字数を分離し、入力中にそれぞれ確認できるようにしました。

## 現在の実装

- **本文 / 構成 / メモ**
  - 本文画面は、左アウトライン・中央本文・右メモの三領域
  - 構成画面は、番号・項目名・本文冒頭・文字数を比較
  - メモには、現在の項目・原稿全体・次に書くことを保存
- **文字数の分離**
  - 原稿タイトルの文字数をタイトル下に表示
  - 各見出しの文字数を見出し行の右端に表示
  - 本文合計は本文欄だけを集計
  - タイトルと見出しの文字数は本文合計へ含めない
  - 空白・改行は文字数へ含めない
  - タイトル、見出し、本文の入力へリアルタイムに追従
- **実用密度**
  - 本文幅を最大46emへ拡張
  - 本文は17px、行間1.72を基準に、一画面の可視行数を増加
  - アウトライン初期幅248px、メモ幅260px
  - 1440px未満ではメモを初期状態で閉じ、本文幅を優先
- **紙とインクの素材設計**
  - 全体は温かな生成りの紙色
  - 本文は青黒い万年筆インクを思わせる色
  - タイトルと見出しは端末内の明朝系書体を優先
  - 真鍮色は番号や選択状態など、補助的な箇所だけに使用
  - 臙脂色はフォーカスや注意が必要な操作へ限定
  - 原稿用紙の格子は本文の背後を避け、余白にだけ極薄く表示
  - 外部画像やWebフォントを使用せず、CSSだけで質感を表現
  - 強制カラーモードと印刷時は質感を自動で取り除く
- **簡素なアウトライン**
  - 開閉・番号・項目名だけを左から配置
  - 選択色やカード状の背景を使わず、太さと一本の線で現在位置を表示
  - 最上位は `1`、子は `1.1`、孫は `1.1.1`
  - 移動・階層変更後に番号を自動再計算
  - 素材感の追加後もアウトラインは無彩色を維持
- **本文見出し番号**
  - 本文の見出しにも `1 / 1.1 / 1.1.1` を表示
  - 原稿上部の「見出し番号」で表示・非表示を切り替え
  - 設定は原稿ごとに保存
- **入力と操作**
  - `Enter`で段落内改行
  - `Ctrl / Cmd + Enter`で次の段落を作成
  - `Tab / Shift + Tab`で階層変更
  - `Alt + Shift + ↑ / ↓`で上下移動
  - 段落の三点記号を廃止し、意味の読める「操作」ボタンを表示
- **メモ表示**
  - 十分な画面幅では右側に常駐
  - 上部の「メモ」ボタンで開閉
  - 狭い画面では右側オーバーレイ、スマホでは下部シート
- **集中表示・折り畳み・未配置**
- **AIとの往復**
  - 選択範囲、段落、部分木、本全体をコピー
  - 本文のみ、見出し付き、Markdown、AI用の形式
- **取り込みと書き出し**
  - プレーンテキスト、Markdown、myspines JSON
  - Markdown、テキスト、完全JSONの書き出し
- **端末内保存**
  - IndexedDBへ自動保存
  - アカウント、クラウド同期、外部送信なし

## ショートカット

| 操作 | キー |
|---|---|
| 段落内で改行 | `Enter` |
| 次の段落を作る | `Ctrl / Cmd + Enter` |
| 一段深くする | `Tab` |
| 一段浅くする | `Shift + Tab` |
| 上下へ移動 | `Alt + Shift + ↑ / ↓` |
| 本文 / 構成 | `Ctrl / Cmd + Shift + O` |
| メモを開閉 | `Ctrl / Cmd + Shift + M` |
| 形式を選んでコピー | `Ctrl / Cmd + Alt + C` |
| 集中 / 一つ上へ戻る | `Ctrl / Cmd + . / ,` |
| Undo / Redo | `Ctrl / Cmd + Z / Shift + Z` |

## UI v4.4の原則

- 本文合計とメタ情報の文字数を混ぜない
- 文字数は入力対象の近くへ置き、執筆の視線を大きく移動させない
- 実用性を損なう装飾は本文の背後へ置かない
- 紙の質感は背景、インクの質感は文字色、文具の高級感は細い境界と控えめな艶で表現する
- アウトラインは無彩色の索引とし、装飾ではなく番号と位置で構造を示す
- 本文の見出し番号は、用途に応じて表示・非表示を選べる
- 明朝体はタイトルと見出しに限定し、本文は読み慣れたゴシック体を維持する
- 原稿データを自動で正規化しない
- 保存失敗だけを明確に伝える

## 保存について

原稿はブラウザのIndexedDBへ保存されます。myspinesが本文をサーバーやAIへ送信することはありません。
ブラウザデータを削除すると原稿も失われる可能性があるため、節目ごとに「完全JSONを退避する」を利用してください。
