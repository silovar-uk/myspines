# myspines

> 段落を動かしながら、長い文章を書くためのローカルファーストエディタ。

myspinesは、本文・番号付きアウトライン・メモを一つの画面で扱う長文執筆ツールです。
UI v4.1では、論文型の番号体系と常駐メモを維持しながら、本文の可視範囲を広げる実用密度へ調整しました。

## 現在の実装

- **本文 / 構成 / メモ**
  - 本文画面は、左アウトライン・中央本文・右メモの三領域
  - 構成画面は、番号・項目名・本文冒頭・文字数を比較
  - メモには、現在の項目・原稿全体・次に書くことを保存
- **実用密度**
  - 本文幅を最大46emへ拡張
  - 本文は17px、行間1.72を基準に、一画面の可視行数を増加
  - アウトライン初期幅248px、メモ幅260px
  - 1440px未満ではメモを初期状態で閉じ、本文幅を優先
  - タイトル・パンくず・見出し・段落の上下余白を圧縮
- **論文型アウトライン**
  - 最上位は `1`、子は `1.1`、孫は `1.1.1`
  - 移動・階層変更後に番号を自動再計算
  - 本文アウトライン、パンくず、構成画面、メモで同じ番号を使用
- **入力**
  - `Enter`で段落内改行
  - `Ctrl / Cmd + Enter`で次の段落を作成
  - `Tab / Shift + Tab`で階層変更
  - `Alt + Shift + ↑ / ↓`で上下移動
- **メモ表示**
  - 十分な画面幅では右側に常駐
  - 上部の「メモ」ボタンで開閉
  - 狭い画面では右側オーバーレイ、スマホでは下部シート
- **集中表示・折り畳み・未配置**
- **AIとの往復**
  - 選択範囲、段落、部分木、本全体をコピー
  - 本文のみ、見出し付き、Markdown、AI用の形式
- **取り込みと書き出し**
  - プレーンテキスト、Markdown、myspines JSON
  - Markdown、テキスト、完全JSONの書き出し
- **端末内保存**
  - IndexedDBへ自動保存
  - アカウント、クラウド同期、外部送信なし

## ショートカット

| 操作 | キー |
|---|---|
| 段落内で改行 | `Enter` |
| 次の段落を作る | `Ctrl / Cmd + Enter` |
| 一段深くする | `Tab` |
| 一段浅くする | `Shift + Tab` |
| 上下へ移動 | `Alt + Shift + ↑ / ↓` |
| 本文 / 構成 | `Ctrl / Cmd + Shift + O` |
| メモを開閉 | `Ctrl / Cmd + Shift + M` |
| 形式を選んでコピー | `Ctrl / Cmd + Alt + C` |
| 集中 / 一つ上へ戻る | `Ctrl / Cmd + . / ,` |
| Undo / Redo | `Ctrl / Cmd + Z / Shift + Z` |

## UI v4.1の原則

- 主役は本文。アウトラインとメモは、本文幅を圧迫しない範囲で表示する
- 一画面に見える文章量を増やしつつ、行長は46字前後を上限にする
- 階層はインデントだけに頼らず、`1 / 1.1 / 1.1.1`で明示する
- 大画面では三領域を同時に見せ、狭い画面では本文幅を優先する
- 操作の強さを、色ではなく位置・境界線・余白・番号で表現する
- ボタンやカードを増やさず、選択中の操作だけを近くに表示する
- 原稿データを自動で正規化しない
- 保存失敗だけを明確に伝える

## 保存について

原稿はブラウザのIndexedDBへ保存されます。myspinesが本文をサーバーやAIへ送信することはありません。
ブラウザデータを削除すると原稿も失われる可能性があるため、節目ごとに「完全JSONを退避する」を利用してください。
